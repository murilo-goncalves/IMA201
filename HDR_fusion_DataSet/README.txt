Instructions
------------

This file contains several registered three-exposure image sets that can be used to evaluate exposure fusion algorithms.

All photos by Ioannis Merianos (ioannismeri@gmail.com)

Please feel free to use this dataset by referencing the paper below.

Merianos I., Mitianoudis N., "A Hybrid Multiple Exposure Image Fusion Approach for HDR Image Synthesis", IEEE International Conference on Imaging Systems and Techniques (IST 2016), Chania, Greece, October 2016.


Ioannis Merianos, Nikolaos Mitianoudis. Democritus University of Thrace, 2016.
